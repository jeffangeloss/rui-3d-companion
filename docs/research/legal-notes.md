# Legal & content notes

## Nature of the project

- **Private, non-commercial fan project** built for personal use (for Gaby).
- Inspired by a theatrical/inventor character archetype. It is **not** affiliated
  with, endorsed by, or representative of SEGA, Colorful Palette, Crypton, or any
  official Project SEKAI entity, and never claims to be.

## What is NOT in this repository

This repo contains **only original source code** and reference documentation. It
does **not** include, and must never include:

- Official Project SEKAI assets (sprites, card art, Live2D, 3D models, stamps).
- Official images, audio, music, MVs, or video.
- 3D models extracted from the game.
- Official voice lines, or any **voice cloning** of the seiyuu / voice actor.
- Data obtained via disallowed scraping.
- Unlicensed fan-made assets.

## What IS allowed

- Original code authored for this project.
- Open-source libraries (under their own licenses).
- Original assets created with Blender / VRoid Studio, or VRM files with a
  compatible license.
- Official material used **only as reference** for tone, personality and visual
  design direction — not copied or redistributed.
- Open-source / clearly-licensed TTS voices (e.g. Piper voices).

## Private asset handling

Private/local assets live in gitignored folders and are never versioned:

```
public/models/private/
public/audio/private/
public/references/
```

## Recommendations

- Keep the GitHub repository **private**.
- Never commit `.env` / `.env.local` or any API keys.
- If the repo is ever made public, the README must state clearly that no official
  assets are included.
